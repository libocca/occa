### Example: Arrays

Example using `occa::array` with inline lambdas

### Compiling the Example

```bash
make
```

### Usage

```
> ./main --help

Usage: ./main [OPTIONS]

Example using occa::array with inline lambdas

Options:
  -d, --device     Device properties (default: "{mode: 'Serial'}")
  -h, --help       Print usage
  -v, --verbose    Compile kernels in verbose mode
```
